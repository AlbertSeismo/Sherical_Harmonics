SHCoeffs

Subclasses
SHRealCoefficients Real spherical harmonic coefficient class.
SHComplexCoefficients Complex spherical harmonic coefficient class.
Initialization
x = SHCoeffs.from_array() Initialize using coefficients from an array.
x = SHCoeffs.from_random() Initialize using random coefficients with a prescribed power spectrum.
x = SHCoeffs.from_file() Initialize using coefficients from a file.
Class methods
x.get_degrees() Return the array [0,...,lmax] that contains the ordered degrees l.
x.get_powerperdegree() Return the power per degree spectrum.
x.get_powerperband() Return the power per logbandwidth spectrum.
x.get_coeffs() Return complex or real spherical harmonics coefficients in different normalizations.
x.rotate() Rotate the spherical harmonics coefficients by [alpha, beta, gamma].
x.expand() Evaluate the coefficients on a spherical grid. Available grid types:
kind ='DH1': equidistant lat/lon grid with nlat=nlon
kind ='DH2': equidistant lat/lon grid with 2*nlat=nlon
kind ='GLQ': Gauss Legendre Grid
x.plot_powerperdegree() Plot the power per degree spectrum.
x.plot_powerperband() Plot the power per logbandwidth(degree) spectrum.
x.make_complex() Convert the real coefficient class to the complex harmonic coefficient class.
x.make_real() Convert the complex coefficient class to the real harmonic coefficient class.

SHGrid

Subclasses
DHGrid Class for Driscoll and Healy (1994) sampled grids.
GLQGrid Class for Gauss-Legendre quadrature sampled grids.
Initialization
x = SHGrid.from_array() Initialize using an array.
x = SHGrid.from_file() Initialize using an array from a file.
Methods
x.get_lats() Return a vector containing the latitude of each row in the grid.
x.get_lons() Return a vector containing the longitude of each column in the grid.
x.plot_rawdata() Plot the grid using a two dimensional cylindrical projection.
x.expand() Expand the gridded data into spherical harmonics.

SHWindow

Subclasses
SymmetricWindow Class for windows concentrated within a spherical cap.
AsymmetricWindow Class for windows concentrated in an arbitrary domain.
Initialization
x = SHWindow.from_cap() Construct windows concentrated with a spherical cap.
x = SHWindow.from_mask() Construct windows concentrated with an arbitrary region.
Methods
x.plot() Plot the best concentrated windows.
x.get_spectrum() Return the power spectrum of the localization windows.
x.get_couplingmatrix() Return the coupling matrix of the first nwin tapers.
x.plot_couplingmatrix() Plot the coupling strength of the localization windows.
x.info() Print meta information about the tapers.
